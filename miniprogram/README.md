# 咱家的记忆 - 微信小程序

一个基于微信小程序的现代化相册管理应用，支持照片浏览、分享和管理功能。

## ✨ 功能特性

### 📱 用户端功能
- **相册浏览**: 瀑布流展示相册列表，支持自定义封面
- **照片管理**: 网格布局展示照片，支持缩略图预览
- **智能排序**: 支持自定义排序、时间排序、文件名排序等多种方式
- **照片预览**: 原生微信图片预览，支持放大缩小、左右滑动
- **保存分享**: 支持保存到系统相册、分享给朋友
- **下拉刷新**: 支持下拉刷新和上拉加载更多
- **详情查看**: 长按照片查看详细信息（文件名、尺寸、大小等）

### 🎨 界面设计
- **现代化UI**: 渐变色彩、圆角卡片、优雅动画
- **响应式布局**: 适配不同屏幕尺寸，完美支持横竖屏
- **流畅动画**: 页面切换、加载状态都有流畅的动画效果
- **用户体验**: 空状态、错误状态、加载状态的友好提示

### 🔧 技术特色
- **API集成**: 与后端FastAPI完美集成
- **性能优化**: 图片懒加载、分页加载、数据缓存
- **错误处理**: 完善的网络错误处理和重试机制
- **状态管理**: 统一的状态管理和数据流

## 🚀 快速开始

### 环境要求
- 微信开发者工具
- Node.js (可选，用于预处理)
- 后端API服务 (FastAPI)

### 安装步骤

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd photo-wxa
   ```

2. **配置API地址**
   打开 `app.js`，修改 `apiBaseUrl` 为你的后端API地址：
   ```javascript
   globalData: {
     apiBaseUrl: 'http://your-api-domain.com/api',
     imageBaseUrl: 'http://your-api-domain.com'
   }
   ```

3. **导入微信开发者工具**
   - 打开微信开发者工具
   - 选择"导入项目"
   - 选择项目目录
   - 填写AppID（测试可以使用测试号）

4. **预览和调试**
   - 在开发者工具中预览
   - 真机调试或发布体验版

## 🔧 开发说明

### 代码结构
- **app.js**: 小程序入口，全局配置和工具方法
- **pages/**: 页面文件，每个页面包含 .js/.wxml/.wxss
- **images/**: 静态图片资源
- **utils/**: 工具函数（如需要）

### 关键功能

#### 1. 图片预览
使用微信原生 `wx.previewImage` API：
```javascript
wx.previewImage({
  urls: imageUrls,          // 图片URL数组
  current: currentImageUrl  // 当前图片URL
})
```

#### 2. 保存到相册
```javascript
// 先下载图片
const res = await wx.downloadFile({ url: imageUrl })
// 保存到系统相册
await wx.saveImageToPhotosAlbum({ filePath: res.tempFilePath })
```

#### 3. 分页加载
```javascript
// 触底加载更多
onReachBottom() {
  if (this.data.hasMore && !this.data.loading) {
    this.loadMorePhotos()
  }
}
```

#### 4. 排序功能
支持多种排序方式：
- `default`: 自定义排序（拖拽设置）
- `created_at_desc`: 最新上传
- `created_at_asc`: 最早上传
- `filename_asc`: 文件名A-Z
- `filename_desc`: 文件名Z-A

## 📞 技术支持

### 常见问题

**Q: 图片加载失败怎么办？**
A: 检查图片URL是否正确，确保网络连接正常，后端服务可访问。

**Q: 保存图片失败？**
A: 确保用户已授权相册权限，可引导用户到设置页面开启权限。

**Q: 请求接口超时？**
A: 检查网络连接，确认后端服务正常，可适当增加超时时间。
