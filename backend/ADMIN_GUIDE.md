# 小宇相册管理后台使用指南

## 🎯 系统概述

小宇相册管理后台是一个功能完整的 Web 管理界面，专门用于管理照片和相册。它与小程序配合使用：

- **管理后台**: 用于上传、管理照片和相册（管理员使用）
- **微信小程序**: 用于查看和浏览照片（用户使用）

## 🚀 快速开始

### 访问地址
- **管理后台**: http://localhost:8000/admin/
- **API 文档**: http://localhost:8000/docs

### 功能导航
- **首页** - 概览统计和快速操作
- **相册管理** - 创建、编辑和管理相册
- **照片管理** - 浏览、编辑和删除照片
- **批量上传** - 高效批量上传照片

## 📊 首页概览

首页提供了系统的整体概览：

### 统计信息
- **相册总数** - 显示系统中的相册数量
- **照片总数** - 显示系统中的照片数量

### 最新内容
- **最新相册** - 显示最近更新的5个相册
- **最新照片** - 显示最近上传的照片缩略图

### 快速操作
- 一键跳转到各功能模块
- 直接访问 API 文档

## 📁 相册管理

### 查看相册列表
- 卡片式展示所有相册
- 显示相册名称、描述、照片数量
- 显示创建时间

### 🆕 相册排序
点击右上角"排序"按钮可按以下方式排序：
- **名称排序**: A-Z / Z-A
- **创建时间**: 最新 / 最早
- **更新时间**: 最新 / 最早（默认）

### 创建新相册
1. 点击右上角"新建相册"按钮
2. 填写相册名称（必填）
3. 填写相册描述（可选）
4. 点击"创建"按钮

### 🆕 编辑相册
1. 点击相册卡片右上角的三点菜单
2. 选择"编辑"选项
3. 修改相册名称和描述
4. 点击"保存"按钮

### 🆕 删除相册
1. 点击相册卡片右上角的三点菜单
2. 选择"删除"选项
3. 确认删除操作
4. ⚠️ **注意：删除相册会同时删除所有照片，不可恢复！**

### 查看相册详情
1. 点击相册卡片上的"查看"按钮
2. 进入相册详情页
3. 查看该相册下的所有照片
4. 可以直接在此页面上传照片

## 📷 照片管理

### 浏览照片
- 网格式展示所有照片
- 显示照片缩略图、文件名、尺寸
- 🆕 显示照片描述（如果有）
- 点击照片可查看大图

### 🆕 照片排序
在相册详情页和照片管理页都可以排序：
- **上传时间**: 最新 / 最早（默认）
- **文件名**: A-Z / Z-A
- **文件大小**: 大到小 / 小到大

### 查看照片详情
点击照片打开详情模态框：
- 查看原图
- 🆕 查看照片描述
- 查看文件信息（文件名、尺寸、大小、上传时间）
- 跳转到所属相册
- 下载原图

### 🆕 编辑照片描述
1. 在照片卡片上点击编辑按钮（铅笔图标）
2. 或在照片详情模态框中点击"编辑描述"
3. 在文本框中输入或修改描述
4. 点击"保存"按钮

### 删除照片
1. 点击照片右下角的删除按钮（垃圾桶图标）
2. 确认删除操作
3. 系统会同时删除原图和缩略图

## 📤 批量上传

### 上传流程
1. 访问"批量上传"页面
2. 选择目标相册
3. 选择照片文件（支持多选）
4. 预览选中的照片
5. 点击"开始上传"

### 支持格式
- **图片格式**: JPG、JPEG、PNG、GIF、WebP
- **文件大小**: 单个文件最大 10MB
- **数量限制**: 一次最多上传 20 张

### 快速上传
在相册详情页面也可以快速上传：
1. 点击右上角"上传照片"按钮
2. 选择文件并预览
3. 直接上传到当前相册

## 🔧 使用技巧

### 1. 高效管理
- 先创建相册，再分类上传照片
- 使用描述功能为相册和照片添加说明
- 🆕 利用排序功能快速找到内容
- 定期整理和删除不需要的照片

### 2. 批量操作
- 利用批量上传功能提高效率
- 一次选择多张相关照片上传
- 按相册主题分类管理

### 3. 🆕 内容组织
- **相册命名**: 使用清晰的命名规则
- **描述编写**: 为相册和照片添加有意义的描述
- **排序使用**: 根据需要选择合适的排序方式
- **定期维护**: 及时更新描述和删除过时内容

### 4. 预览和确认
- 上传前仔细预览选中的照片
- 确认照片质量和内容适合
- 删除前再次确认避免误删

## 📱 与小程序对接

### 数据同步
- 管理后台的所有操作会实时同步到小程序
- 新上传的照片立即在小程序中可见
- 删除的照片会从小程序中移除
- 🆕 照片描述会在小程序中显示

### 建议工作流程
1. **内容规划** - 确定要展示的照片主题
2. **相册创建** - 在管理后台创建对应相册并添加描述
3. **照片上传** - 批量上传高质量照片
4. **🆕 内容编辑** - 为照片添加描述信息
5. **🆕 内容排序** - 调整展示顺序
6. **内容审核** - 在小程序中查看展示效果
7. **优化调整** - 根据需要删除或添加照片

## ⚠️ 注意事项

### 文件限制
- 确保照片文件格式正确
- 注意单文件大小限制（10MB）
- 避免上传过多文件导致超时

### 操作安全
- 删除操作不可恢复，请谨慎操作
- 🆕 删除相册会同时删除所有照片

### 性能建议
- 批量上传时建议分批进行
- 避免上传过大的照片文件
- 定期清理不需要的照片
- 🆕 合理使用描述功能，避免过长文本
